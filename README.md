# 🛒 Умная корзина

Оптимизатор семейных закупок — веб-приложение, которое помогает разбить список покупок по разным торговым точкам так, чтобы потратить меньше денег.

**Демо:** https://sanprog.github.io/smart-basket/

## Суть приложения

Цены на одни и те же товары в гипермаркете, на базаре и в магазине у дома отличаются на 10–40%. Обычно покупатель либо переплачивает, покупая всё в одном месте, либо тратит время на ручное сравнение цен.

«Умная корзина» решает эту задачу: пользователь отмечает нужные товары и указывает количество, а приложение само:

- находит для каждого товара точку с минимальной ценой;
- группирует товары в отдельные корзины по точкам с подытогами;
- считает общую сумму при оптимальном распределении и сравнивает её со сценарием «всё в гипермаркете»;
- показывает итоговую экономию в тенге и в процентах;
- учитывает **порог целесообразности** — если поездка в отдельную точку экономит меньше заданной суммы (по умолчанию 1000 ₸), её товары автоматически переносятся в основную точку (гипермаркет).

Приложение — MVP на тестовых данных (см. [prd.md](prd.md)): 20 товаров в 3 категориях (продукты, бытовая химия, товары для ребёнка) и 3 типа точек (гипермаркет, базар, магазин у дома). Цены встроены в код и приближены к реальным ценам Алматы, но не претендуют на точность.

## Технологии

Чистый HTML/CSS/JavaScript, без фреймворков, без сборки, без бэкенда и БД. Работает полностью в браузере.

- [index.html](index.html) — разметка страницы
- [style.css](style.css) — стили, адаптив для мобильных экранов
- [app.js](app.js) — логика расчёта корзин и рендер интерфейса
- [data.js](data.js) / [data.json](data.json) — справочник товаров и цен

## Как запустить

Самый простой способ — открыть [index.html](index.html) двойным кликом в браузере. Приложение не обращается к серверу и не требует установки зависимостей.

Если нужен локальный веб-сервер (например, для тестирования через инструменты разработки):

```bash
# Python
python -m http.server 8080

# Node.js
npx http-server -p 8080
```

Затем открыть `http://localhost:8080` в браузере.

## Как пользоваться

1. Отметьте нужные товары чекбоксами в справочнике и укажите количество.
2. Справа сразу появится разбивка по корзинам (гипермаркет / базар / магазин у дома) с подытогами.
3. При необходимости измените порог целесообразности — от него зависит, будут ли товары раскиданы по нескольким точкам или собраны в одной.
4. Кнопка «Сбросить список» очищает выбор и возвращает порог к значению по умолчанию.
